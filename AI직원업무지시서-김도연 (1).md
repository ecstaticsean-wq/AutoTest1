## 1. 신청자 맥락

**만들고 싶은 AI 직원**
상품 타게팅 홍보 AI 직원 — 신선식품, 가공식품, 계절가전 상품정보를 받아 타겟 고객층을 분석한 후 쓰레드용 글과 블로그용 글을 작성해 비공개 판매페이지로 고객을 유입시키는 일

**지금 겪는 문제**
- 위탁판매 상품이 항상 바뀌는데, 맞는 고객을 찾아 홍보하는 과정이 반복되고 시간이 많이 걸림
- 기존에 쓰레드, 블로그, 카카오채널, 문자 등으로 직접 홍보했지만 꾸준히 하기 어려움
- 판매채널이 비공개 쇼핑몰로 제한적이어서 노출이 어려움

**왜 필요한가**
- 쓰레드가 가장 반응이 좋았음 (기존 시도 중)
- 상품마다 새로 글을 만들고 채널별로 올려야 하는 반복 시간을 단축하고 싶음
- 채널별 최적화된 콘텐츠로 인입을 증대하고 싶음

## 2. MVP 입력·처리·출력 흐름

### 입력 (신청자가 제공)
- 상품명
- 판매이미지 (1장)
- 대표판매사이트 URL
- 내 판매페이지 URL

### 처리 (AI가 수행)
1. **타겟 고객층 분석**: 상품명과 판매이미지를 보고 주요 구매층이 누구인지 파악
2. **쓰레드용 글 작성**: 타겟층에 맞는 톤으로 쓰레드 글 작성 (대표판매사이트 URL 포함)
3. **블로그용 글 작성**: 타겟층에 맞는 톤으로 블로그 글 작성 (내 판매페이지 URL 포함)
4. **연출이미지 설명 작성**: 블로그 글에 어울리는 연출이미지 1장의 설명 작성 (이미지 생성용 프롬프트)

### 출력 (화면에 표시)
- 타겟 고객층 분석 결과 (1~2문장)
- 쓰레드용 글 (완성본, 복사 가능)
- 블로그용 글 (완성본, 복사 가능)
- 연출이미지 설명 (이미지 생성 도구에 붙여넣을 수 있는 형태)

---

## 3. 오늘 만들 MVP 범위

**목표**: 상품 1개의 정보를 입력받아 쓰레드 글, 블로그 글, 연출이미지 설명이 모두 생성되는 흐름을 완성하기

**구현 방식**
- ChatGPT API를 사용해 텍스트 생성 (쓰레드 글, 블로그 글, 이미지 설명)
- 이미지 생성 도구는 선택사항 (MVP에서는 설명만 제공, 실제 이미지 생성은 신청자가 별도 도구 사용)

**신청자가 직접 테스트할 것**
- 실제 상품 1~2개를 입력해 결과가 화면에 나오는지 확인
- 생성된 글의 톤과 소구포인트가 맞는지 검토
- 1주일 뒤 실제로 쓰레드와 블로그에 올려본 후 반응(클릭, 문의) 측정

## 5. 개발 시 주의사항

### API 키 보안
- ChatGPT API 키는 코드에 직접 쓰지 않기
- `.env` 파일에 저장하고 `process.env` 또는 `os.getenv()`로 불러오기
- `.gitignore`에 `.env` 추가해서 GitHub에 올리지 않기

### 개인정보 보호
- 신청자의 실제 판매페이지 URL은 샘플 데이터로 테스트
- 생성된 글에 신청자의 개인정보나 실제 고객정보가 포함되지 않도록 확인

### 입력값 검증
- 상품명: 빈 값이 아닌지 확인
- 판매이미지: 파일 형식 확인 (jpg, png 등)
- URL: 유효한 URL 형식인지 확인

### 에러 처리
- API 호출 실패 시 사용자에게 명확한 에러 메시지 표시
- 타임아웃 발생 시 재시도 옵션 제공

## 7. 성공 기준

**수업 당일 검증**
- 실제 상품 정보 1개를 입력했을 때 결과가 화면에 나오는가?
- 생성된 쓰레드 글과 블로그 글이 복사 가능한가?
- 연출이미지 설명이 이미지 생성 도구에 붙여넣을 수 있는 형태인가?

**1주일 뒤 검증**
- 생성된 쓰레드와 블로그 글을 실제로 올려봤는가?
- 반응(클릭, 문의)이 있었는가?
- 소구포인트 분석이 맞았는가?

## 8. 개발 진행 순서

1. **입력 폼 만들기**: 상품명, 판매이미지, URL 2개를 받을 수 있는 폼 구성
2. **ChatGPT 프롬프트 작성**: 타겟 고객층 분석 → 쓰레드 글 → 블로그 글 → 이미지 설명을 순차적으로 생성하는 프롬프트
3. **API 연동**: ChatGPT API 호출해서 결과 받기
4. **결과 표시**: 생성된 텍스트를 화면에 표시하고 복사 버튼 추가
5. **테스트**: 실제 상품 정보로 1회 테스트해서 결과 확인

## 9. 참고: 신청자 대화 내용 정리

- **상품군**: 신선식품(농산물), 가공식품(커피/차/디저트류), 계절가전
- **판매채널**: 비공개 쇼핑몰 (오픈마켓 제외)
- **기존 시도**: 기존고객 문자, 카카오채널, 블로그, 쓰레드
- **가장 효과 있는 채널**: 쓰레드
- **쓰레드 글**: 텍스트만
- **블로그 글**: 텍스트 + 연출이미지 설명
- **핵심 요청**: 상품명과 대표판매사이트 URL을 넣으면 타겟층을 분석하고 그에 맞게 글을 써서 내 판매페이지로 인입시키는 AI 직원
